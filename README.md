## How to Run ▶
**Clone**
```shell
git clone https://github.com/Chen-Huayi/Membership-Management.git
```
**Run**
+ In `/Membership-Management/backend` directory:
```shell
npm install
npm run server
```
+ In `/Membership-Management/frontend` directory:
```shell
npm install
npm run start
```
